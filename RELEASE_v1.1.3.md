# AVTOKarta v1.1.3

**Обновление безопасности и исправление системы обновлений**

---

## Исправления

### Система обновлений
- Исправлено скачивание обновлений — заменён `WebClient` на `HttpWebRequest` для корректной обработки редиректов GitHub
- `.exe` установщики теперь запускаются тихо (`/SILENT /VERYSILENT`) вместо ручной замены файлов
- Добавлена SHA-256 проверка скачиваемых файлов через `digest` из GitHub API
- Полное экранирование спецсимволов (`^`, `!`, `%`, `&`, `|`, `<`, `>`, `(`, `)`) в `.bat` скриптах обновления
- Сообщения в `.bat` скриптах заменены на латиницу — устранён битый текст при ошибках

### Безопасность шифрования
- PBKDF2 переведён с SHA-1 на SHA-256 (300 000 итераций) — ручная реализация для совместимости с .NET 4.5.2
- Кеш ключей шифрования заменён на `ConcurrentDictionary` — потокобезопасность при параллельных операциях
- `EncryptionService` реализует `IDisposable` — пароль и кеш ключей очищаются из памяти

### Защита данных
- Защита от zip-бомб при импорте: лимит 50 МБ на запись, 10 000 записей, общий объём до 500 МБ
- Стек-трейсы скрыты от пользователя — показывается только сообщение об ошибке
- Каталог `crash.log` ограничен правами текущего пользователя (ACL) при каждом запуске

### Интерфейс
- Добавлена прокрутка на экран первоначальной настройки (стартовый экран)
- Добавлен скроллбар в боковую панель навигации — при малом размере окна кнопки прокручиваются
- Исправлено отображение иконки приложения — теперь корректно отображается в проводнике, на рабочем столе, в ярлыках и в панели задач

### Техническое
- Добавлен `Accept` заголовок и лимит редиректов для GitHub API
- Добавлено свойство `ExpectedHash` в `UpdateResult` для хранения SHA-256 хеша
- `TypeNameHandling.None` установлен во всех вызовах `JsonConvert.DeserializeObject`
- Path traversal защита в `DataTransferService` усилена двойной проверкой
