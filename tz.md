#### 2.3.3. Формулы расчёта

**Расход топлива по норме для одной записи:**

```
По_норме = (Пробег_км × Норма_пробега_без_насоса)
         + (Время_с_насосом_мин × Норма_насоса)
         + (Время_без_насоса_мин × Норма_холостого_хода)
         + (Смена_караула_мин × Норма_смены)
         + (Прочие_работы_мин × Норма_прочих)
```

**Фактический расход:**

```
Если Заправлено_за_месяц = 0:
    Факт = По_норме
Иначе:
    Факт = Остаток_нач + Заправлено_за_месяц - Остаток_конец
```

**Итоги за месяц:**

```
Итого_факт = Σ(Факт_по_всем_записям)
Итого_норма = Σ(Норма_по_всем_записям)
Экономия = max(0, Итого_норма - Итого_факт)
Перерасход = max(0, Итого_факт - Итого_норма)
```

**Расход по элементам:**

```
Расход_пробег = Σ(Пробег_км) × Норма_пробега_без_насоса
Расход_с_насосом = Σ(Время_с_насосом_мин) × Норма_насоса
Расход_без_насоса = Σ(Время_без_насоса_мин) × Норма_холостого_хода
Расход_смена = Σ(Смена_караула_мин) × Норма_смены
Расход_прочие = Σ(Прочие_работы_мин) × Норма_прочих
ВСЕГО = Расход_пробег + Расход_с_насосом + Расход_без_насоса
      + Расход_смена + Расход_прочие
```

**Приведённый пробег:**

```
Приведённый_пробег = Σ(Пробег_км)
                   + Σ(Смена_караула_мин + Прочие_работы_мин) × Коэффициент_приведения
```

**Распределение по типам выездов:**

| Ключевые слова | Тип выезда |
|----------------|------------|
| «учени», «отработка», «норматив» | Учебный выезд |
| «пожар», «выезд на пожар» | Пожар |
| «ложно» | Ложно |
| Всё остальное | Прочий пробег |

Для каждого типа: Км = Σ(Пробег_км), Л = Σ(Факт_по_норме)

```
Расход_пробег = Σ(Пробег_км) × Норма_пробега_без_насоса
Расход_с_насосом = Σ(Время_с_насосом_мин) × Норма_насоса
Расход_без_насоса = Σ(Время_без_насоса_мин) × Норма_холостого_хода
Расход_смена = Σ(Смена_караула_мин) × Норма_смены
Расход_прочие = Σ(Прочие_работы_мин) × Норма_прочих
ВСЕГО = Расход_пробег + Расход_с_насосом + Расход_без_насоса
      + Расход_смена + Расход_прочие
```

### 7.5. Приведённый пробег

```
Приведённый_пробег = Σ(Пробег_км)
                   + Σ(Смена_караула_мин + Прочие_работы_мин) × Коэффициент_приведения
```

Коэффициент_приведения по умолчанию: **0.35 км/мин**

### 7.6. Распределение по типам выездов

| Ключевые слова в наименовании | Тип выезда |
|-------------------------------|------------|
| «учени», «отработка», «норматив» | Учебный выезд |
| «пожар», «выезд на пожар» | Пожар |
| «ложно» | Ложно |
| Всё остальное | Прочий пробег |

Для каждого типа:
- **Км** = Σ(Пробег_км) по записям этого типа
- **Л** = Σ(Факт_по_норме) по записям этого типа
